# dashbouton
Dans ce projet nous allons creer un dashbouton capable de rajouter ou d'enlever un produit dans un compteur.
## liste des composant

+ un arduino esp3221
+ deux boutons pushboutton
+ 1 breadboard
+ deux resistance 220oms
+ des cables male/femelle
